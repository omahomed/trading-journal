Weekly Retro — v1 (LOCKED)

Reference design for mobile-weekly-retro.tsx (Phase 2 T2-2). Read-only list of weekly retrospectives scoped to the active portfolio. Replaces the desktop 706-LOC monolith (rail + tiles + form + modal) for mobile viewports.
  • Tap a card → /weekly-retro?week=YYYY-MM-DD (desktop detail view; gets its own mobile-fit in T2-5 after the 5 MB cap bump and T2-4 land)
  • Filters: Year + Month pills (default current-current → ~3-5 cards). Reset chip snaps both back. Cross-month weeks filter by week_start month.
  • Per-card fields: pin (if pinned) · week span · grade pill · chevron / weekly return % · weekly P&L $ / N trades · LTD% · YTD% / tag pills (conditional, omitted when empty)
  • LTD/YTD computed client-side from chained sparkline_value (weekly_return_pct). LTD spans all years; YTD resets at Jan 1.
  • Tag colors sourced from per-tag stored TAG_PALETTE key (rose | amber | emerald | sky | violet) — falls back to neutral muted on unknown key.
  • Synthetic empty Mondays filtered out (desktop rail needs them for sparkline continuity; the mobile list is real retros only).
