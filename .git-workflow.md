# Git Workflow Quick Reference

## Daily Workflow (After Editing app.py)

```bash
# 1. See what changed
git status

# 2. Save changes
git add app.py
git commit -m "Brief description"
```

## View History

```bash
# See all versions (newest first)
git log --oneline

# See last 5 commits
git log --oneline -5

# See what changed in a commit
git show <commit-id>
```

## Compare Changes

```bash
# See current uncommitted changes
git diff app.py

# Compare to previous version
git diff HEAD~1 app.py

# Compare to 3 versions ago
git diff HEAD~3 app.py
```

## Undo/Restore

```bash
# Undo uncommitted changes (BE CAREFUL!)
git checkout -- app.py

# Restore to a previous version
git checkout <commit-id> app.py

# Go back to latest
git checkout main app.py
```

## Useful Aliases (Optional - Add to ~/.gitconfig)

```bash
git config --global alias.lg "log --oneline --graph --all"
git config --global alias.st "status -sb"
git config --global alias.save "!git add -A && git commit -m"
```

Then you can use:
- `git lg` - Pretty log
- `git st` - Short status
- `git save "message"` - Quick save all changes
